# Healthcare-Analytics-Project
County Referral Hospital Dashboard Data analysis dashboard using PowerBi
A Power BI dashboard project analysing hospital pharmacy data across six counties in Kenya. The dashboard provides interactive insights into drug costs, patient visits, disease patterns, and medication usage to support data-driven decision making in hospital management.

##  Dashboard Overview

The **County Referral Hospital Dashboard** visualises key metrics from hospital pharmacy records, allowing users to filter and explore data by county, diagnosis, age group, and gender.

**Live KPIs tracked:**
- Total Drug Costs — **275K KES**
- Average Length of Stay — **1.13 days**
- Total Outpatient Visits — **95**
- Total Inpatient Visits — **104**
- Total Emergency Visits — **101**
- Total Hospital Visits — **300**

## Key Visuals in the Dashboard

| KPI Cards | Summary of total visits, drug costs, and length of stay |
| Medication Usage by Age Group | Horizontal bar chart showing drug usage across age bands |
| Diagnosis Trend by Month | Line chart tracking disease prevalence over time |
| Pharmacy Costs by Department | Pie chart breaking down costs by Emergency, Inpatient, Outpatient |
| Common Diseases by County | Matrix table showing disease counts across 6 counties |
| Department by County | Stacked bar chart comparing department visits per county |
| Drug Category Costs | Bar chart showing costs per drug/disease category |

## Filters / Slicers

The dashboard includes the following interactive filters:
- **Diagnosis** — Diabetes, Flu, Hypertension, Malaria, Pneumonia, Typhoid
- **County** — Kiambu, Kisumu, Mombasa, Nairobi, Nakuru, Uasin Gishu
- **Age Group** — 0–17, 18–25, 26–40, 41–60, 60+
- **Gender** — Female, Male

## Key Insights

1. **Typhoid** is the most expensive drug category at 62K KES, followed by Hypertension and Pneumonia at 46K KES.
2. The **60+ age group** consumes the most medication, highlighting the need for elderly-focused care planning.
3. **Inpatient** services account for 34.41% of total pharmacy costs, the highest of all departments.
4. **Flu** has the highest total case count (56), followed by Typhoid at (54) across all counties.
5. **Kiambu** records the highest Typhoid burden (7 cases), suggesting potential water/sanitation concerns.
6. Hospital visits are evenly distributed across Inpatient, Outpatient, and Emergency departments.
